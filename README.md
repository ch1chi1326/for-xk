Интерактивная карта моего города

Добро пожаловать на проект Интерактивная карта моего города— статический сайт, опубликованный с помощью GitHub Pages.

На этой карте вы можете увидеть разные объекты города, такие как:

- Парки  
- Школы  
- Политические места

 Обзор

Этот сайт представляет собой интерактивную карту с важной информацией о различных объектах городской инфраструктуры.  
Цель проекта — визуализировать основные точки интереса в городе в одном месте
 Запуск и использование

1. Откройте сайт по адресу:  
   https://ch1chi1326.github.io/for-xk/  
2. На странице вы увидите карту с отмеченными объектами: парки, школы и политические точки интереса.  
3. Используйте карту для изучения и навигации

Проект создан с использованием:

- *HTML* — структура страницы  
- *CSS* — стилизация  
- *JavaScript* — логика интерактивной карты  
- *GitHub Pages* — хостинг сайта
